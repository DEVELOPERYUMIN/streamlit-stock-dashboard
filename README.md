# 📊 Streamlit 주가 조회 대시보드

KRX 상장사의 주가 데이터를 조회하고  
**수익률 · 최근 흐름 · 리스크(MDD) · 주요 뉴스 헤드라인**을  
한 화면에서 확인할 수 있는 Streamlit 기반 주가 분석 대시보드입니다.

---

## 주요 기능

### 회사 검색 및 주가 조회
- KRX 상장사 목록 기반 회사명 검색
- 회사 선택 시 종목코드 자동 매핑
- 종목코드는 숫자 6자리만 허용(zfill 처리)
- 조회 기간(시작일 / 종료일) 선택 후 주가 데이터 조회

---

### 요약 지표 카드
- 최근 종가
- 기간 수익률
- 최근 흐름 요약
  - 1주 / 1개월 / 3개월 수익률
  - 상승(🔺) / 하락(🔻) 방향 표시
- 최고가 / 최저가 (종가 기준)
- 최대낙폭(MDD, Max Drawdown)
- 변동성(일간 수익률 표준편차)

---

### 인터랙티브 차트
- 종가(Close) 라인 표시
- 이동평균선 선택 표시
  - MA20 / MA60
- 거래량(Volume) 보조축(y2) 표시
- hovermode = x unified
  - 같은 날짜의 종가 / MA / 거래량을 한 번에 확인 가능

---

### 최대낙폭(MDD) 시각화
- 기간 중 최대낙폭이 발생한 고점 → 저점 구간 자동 계산
- 해당 구간을 붉은 음영으로 강조 표시
- Peak / Trough 마커 및 라벨 표시
- 조회 기간 변경 시 MDD 구간 재계산

---

### 종가 타임-플레이 애니메이션
- 종가 라인을 시간 순서대로 점진적으로 재생
- 최대 260 프레임으로 제한하여 성능 고려
- Play / Pause 버튼 제공

---

### 주요 뉴스 헤드라인
- 회사명 기준 Google News RSS 검색
- 뉴스 제목 / 출처 / 발행 시각 표시
- 클릭 시 원문 기사로 이동
- 뉴스 데이터는 10분 캐시 적용

---

### 데이터 테이블 및 엑셀 다운로드
- 최근 주가 데이터 테이블 제공
- 조회 결과를 Excel 파일(.xlsx)로 다운로드 가능

---

## 차트 해석 가이드
앱 내부 페이지로 제공
- 주식 기초
- 차트 읽는 법
- 수익률 / MDD / 이동평균 / 거래량 해석 기준 정리

---

## 이 앱이 하지 않는 것
- 매수/매도 추천 또는 투자 판단 대행
- 실시간(틱 단위) 체결 데이터 제공
- 뉴스·재무제표 기반 자동 투자 판단

---

## 사용 기술
- Python
- Streamlit
- FinanceDataReader
- Plotly
- Pandas
- Google News RSS

---

## 데이터 기준
- 주가 데이터: 일봉 기준
- 수익률 / MDD / 변동성 계산은 종가 기준
- 배당, 세금, 수수료는 반영하지 않음
